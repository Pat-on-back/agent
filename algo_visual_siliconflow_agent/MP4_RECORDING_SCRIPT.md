# MP4 录屏讲解稿

大家好，我展示的是“基于硅基流动 API 的通用算法过程可视化智能体”。

这个智能体解决的问题是：很多算法只看文字和代码很难理解，因此我们把算法运行过程拆成逐步可视化帧，让学习者观察状态变化、搜索路径、剪枝过程和迭代收敛。

首先打开公开访问链接。可以看到侧边栏中配置的是硅基流动 API，接口地址为 https://api.siliconflow.cn/v1。项目使用公开可访问的大模型 API，不使用内部未公开模型。

接下来选择 01 背包问题。这里的物品包含重量和价值，背包容量为 15。我勾选回溯法、动态规划、分支限界、遗传算法和模拟退火五种算法，然后点击生成可视化。

现在可以看到：
- 动态规划展示 dp 状态表逐步填充；
- 回溯法展示取或不取的决策路径；
- 分支限界展示上界估计和剪枝；
- 遗传算法展示种群最优值随代数变化；
- 模拟退火展示温度下降、邻域扰动和接受概率。

这说明同一个问题可以用不同算法思想求解：动态规划强调状态转移，回溯强调搜索，分支限界强调剪枝，遗传算法和模拟退火强调启发式迭代优化。

然后我切换到 LCS、N 皇后或最短路径，说明该智能体不是只支持 01 背包，而是一个通用算法过程可视化智能体。

最后点击生成 PDF 报告。PDF 中自动包含问题建模、算法过程、关键帧说明、结果对比和学习总结。
