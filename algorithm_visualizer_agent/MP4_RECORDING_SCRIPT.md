# MP4 录屏讲解稿

大家好，这是我完成的算法过程可视化智能体。这个智能体的目标是帮助学习者通过可视化过程更快理解算法核心思想，而不是只阅读抽象文字或伪代码。

首先可以看到，这是一个公开网页应用，部署后可以通过链接直接访问。系统使用公开可访问的大模型 API，例如 GPT、通义千问或豆包火山方舟，不使用内部未公开模型。

现在我输入一个算法学习问题：请用可视化方式解释快速排序如何把数组 [8, 3, 5, 1, 9, 6, 2, 7] 排序。然后点击开始可视化求解并生成报告。

智能体首先识别问题对应的算法类型，这里识别为快速排序。接着它调用轻量工具生成算法执行轨迹，包括当前区间、pivot、比较指针和交换位置。页面中间展示的是逐帧可视化图，我可以拖动步骤条观察每一步发生了什么。

在快速排序中，最重要的是 pivot 的划分过程。通过动画可以看到，小于等于 pivot 的元素会被交换到左侧，pivot 归位后再递归处理左右子区间。相比只看代码，这种逐帧过程更容易理解算法的核心思想。

右侧显示了智能体自动生成的教学讲解和当前状态变量。下方还可以展开完整步骤表和工具调用记录，说明系统确实经历了算法识别、轨迹生成、可视化渲染和 PDF 生成这些步骤。

最后点击下载 PDF 求解报告。PDF 中包含输入问题、识别结果、核心思想、可视化步骤摘要和工具调用记录。这就是本项目最终提交的 PDF 文档。
