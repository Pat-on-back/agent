# 提交检查清单

## 1. 可访问链接

- [ ] 已上传代码到 GitHub。
- [ ] 已部署到 Streamlit Community Cloud。
- [ ] 打开公开链接后，其他人无需本地运行即可访问网页。
- [ ] API Key 没有写入公开代码，而是放在 Streamlit Secrets 中。

## 2. MP4 视频

录屏内容建议控制在 3-5 分钟。

- [ ] 展示公开访问链接。
- [ ] 说明系统使用公开可访问的大模型 API。
- [ ] 输入算法学习问题。
- [ ] 点击“开始可视化求解并生成报告”。
- [ ] 展示算法过程可视化步骤。
- [ ] 展示状态变量和工具调用记录。
- [ ] 下载并打开 PDF 报告。

## 3. PDF 文档

- [ ] PDF 是智能体网页自动生成的。
- [ ] PDF 包含输入问题、识别算法、核心思想、步骤摘要和工具调用记录。
- [ ] PDF 文件名清晰，例如：算法过程可视化智能体_求解报告.pdf。

## 4. 推荐演示题目

```text
请用可视化方式解释快速排序如何把数组 [8, 3, 5, 1, 9, 6, 2, 7] 排序。
```

推荐理由：快速排序的 pivot、比较、交换和递归分治过程很直观，适合展示“可视化帮助理解算法核心思想”的项目价值。
