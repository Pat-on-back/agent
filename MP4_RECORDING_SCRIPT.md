# MP4 录屏讲解稿

## 录屏标题

基于硅基流动 API 的算法过程可视化智能体演示

## 录屏顺序

### 1. 展示公开网页

打开部署后的 Streamlit 公开链接，说明项目名称为“算法过程可视化”。

讲解词：

> 这是我实现的算法过程可视化智能体。它不是让用户从固定问题类型中选择，而是直接输入算法题目，由大模型自动识别问题并生成可视化过程。

### 2. 展示 API 来源

打开侧边栏，展示硅基流动 API 配置项。

讲解词：

> 本项目使用的是硅基流动公开 API。算法识别、过程拆解、可视化帧和教学报告主要由大模型生成，本地程序只负责调用 API、渲染结果和导出 PDF。

### 3. 输入演示题目

推荐输入：

```text
请可视化求解 01 背包问题。背包容量 15，物品 A 重量2 价值6，B 重量3 价值10，C 重量4 价值12，D 重量5 价值14，E 重量9 价值20，F 重量7 价值18。请用回溯法、动态规划、分支限界、遗传算法、模拟退火进行对比。
```

讲解词：

> 这里输入的是自然语言题目，并没有手动选择“01 背包”这个类型。

### 4. 生成可视化

点击“生成算法过程可视化”。等待大模型返回结果。

讲解词：

> 大模型会自动识别问题类型，并把不同算法的过程转化成逐帧展示的数据。

### 5. 展示算法结果

依次展示：

- 回溯法：决策树和剪枝；
- 动态规划：状态表转移；
- 分支限界：上界和剪枝；
- 遗传算法：染色体和适应度变化；
- 模拟退火：温度下降和接受概率。

讲解词：

> 这些图和表不是本地算法工具预先写死的，而是由大模型根据题目生成的可视化帧。

### 6. 展示 PDF 下载

点击“下载 PDF 报告”，打开生成的 PDF。

讲解词：

> 最后，智能体会将算法过程、结果和对比总结生成 PDF，作为提交材料的一部分。

### 7. 总结提交材料

讲解词：

> 本项目最终提交三项内容：公开访问链接、演示 MP4 视频和智能体生成的 PDF 文档。


## PDF 展示补充说明

录屏最后一定要打开下载的 PDF，展示其中包含：自动识别结果、逐算法求解过程、关键变量、可视化帧、算法对比和完整教学报告。这样可以证明“智能体求解过程要自己输出一个详细文档 PDF”。
