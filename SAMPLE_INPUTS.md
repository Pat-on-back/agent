# 示例输入

## 1. 01 背包多算法对比

```text
请可视化求解 01 背包问题。背包容量 15，物品 A 重量2 价值6，B 重量3 价值10，C 重量4 价值12，D 重量5 价值14，E 重量9 价值20，F 重量7 价值18。请用回溯法、动态规划、分支限界、遗传算法、模拟退火进行对比。
```

## 2. N 皇后回溯

```text
请用回溯法可视化求解 4 皇后问题，展示每一步放置、冲突判断和回退过程。
```

## 3. LCS 动态规划

```text
请用动态规划可视化求解字符串 ABCBDAB 和 BDCABA 的最长公共子序列。
```

## 4. Dijkstra 最短路径

```text
请可视化 Dijkstra 算法求解最短路径。图中边为 A-B:4，A-C:2，B-C:1，B-D:5，C-D:8，C-E:10，D-E:2，D-F:6，E-F:3，起点为 A。
```

## 5. 快速排序

```text
请可视化快速排序对数组 [8, 3, 5, 1, 9, 6, 2, 7] 的排序过程，展示 pivot、交换和递归划分。
```
